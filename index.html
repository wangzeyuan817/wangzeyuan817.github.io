前端入门教程之HTML

HTML配置准备

  • 介绍：英文全称 Hyper Text Marked Language ，即超文本标记语言，是一种标识性的语言。它包括一系列标签．通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等。

    • 版本：HTML 5
    • 特点：
      • 简易性：超级文本语言
      • 平台无关性：任何平台都可以应用
      • 通用性：任何浏览器和电脑都可以浏览

  • 推荐开发软件：VS code

    • 下载地址： https://code.visualstudio.com/

    • 插件配置：

      1. 打开VS code，找到安装插件的侧栏并且点击

      2. 在侧栏搜索栏搜索插件：view in browser 和open in browser并点击install安装

      3. 设置中文语言环境（推荐使用英文，适应英文环境）

        1. 按键语言设置快捷键：ctrl+shift+p

        2. 搜索：配置语言或者 Configure Language并点击

          如图：

        3. 选择显示语言：可以选择en为英语，zh-cn为中文，或选择安装其他语言点击install后重复上述操作在选择显示语言栏中选择你需要的语言

          如图：

        4. 重启VS code完成配置

HTML文件创建和操作

  • 创建：

    1. 点击左上角文件/新建文件夹，在文件夹里新建一个文件并命名demo.html，后缀为.html回车，创建成功（注意：文件命名不要用中文，否则后期和CSS共用的时候会很麻烦）

    2. 在新建文件中右键点击open in default browser 可以打开一个网页，此时是空白的

HTML标签

  • 意义：HTML语法基本的组成，用来构成网页的基本结构

  • 区别：双标签和但标签

  • 基本语法：

    • 注释：<!-注释内容->，注释只能放在代码末尾，不能放在代码前部，否则会导致代码错误

      1.

      你的内容

      2.

      正确示范

      • 注释的作用：

        1. 为开发人员解释代码用途

        2. 提示自己代码功能方便他人和自己阅读修改

        3. 标注代码，使代码暂时失效用于调试

    • 标题：

      • title:网页名称
      • h:页面标题，分为h1,h2,h3,h4,h5,h6,六个等级，在网页上显示大小由大到小如图：可以观察到大小不同，h1等级最高，h6等级最低，依次重要性也递减
      • 段落：双标签
        • 语法：

          段落文字

        • 属性
